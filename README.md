# Crypto-Notifier
Desktop notification tool showing the latest crypotcurrency rates built for linux using python. 
<br><br>
**Installation/setup**
1. Run `pip install -r requirements.txt`
2. `cd` into `cryptonotifier`
3. Run `python  Main.py`
<br><br>

**Requirements**
* requests<=2.11.1
* beautifulsoup4
* notify2
